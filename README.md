# Game of Life

Classical game of life implemented using the following technologies:

- TypeScript
- React 18.0.15
- Tailwind CSS
- Vite
- Node.js version 18.4.0, npm version 8.12.1
- HTML Canvas element

Template for the project was created usin create-vite:

    npm create vite@latest game-of-life --template react-ts

Tailwind CSS was added using instructions from [Vite Guide](https://tailwindcss.com/docs/guides/vite).

Formatting of code by Prettier.

## Building

    npm i
    npm run build
    npm run dev

## How to Play
Hold left mouse button down to draw, right mouse button to erase, then start.

You can play it [here](https://cool-html5.net/game-of-life/).
